# Jobly
Job Searching API with React Frontend and Express, Node, PostgreSQL Backend
Deployed on: https://clupt-jobly.surge.sh/

## Local Setup

1. Install dependencies and run app.

    ```
    npm i
    npm start
    ```

Open [http://localhost:3000] to view it in your browser.
